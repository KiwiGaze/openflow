# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind. Maintainers may remove,
edit, or reject contributions and contributors that violate this standard.

Report unacceptable behavior privately to the maintainers via the contact listed on the
repository profile. Reports are handled confidentially.
